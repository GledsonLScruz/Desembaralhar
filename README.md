Desembaralhar ⏳
===================================

Desembaralhar é um game Single Player que mostra palavras embaralhadas. Para jogar, o jogador tem que criar uma palavra usando todas as letras que foram mostradas.

Conhecimentos:
====================================
ViewModel,
LiveData,
DataBinding.
